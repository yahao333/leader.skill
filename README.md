# leader.skill

蒸馏领导者的愿景、决策风格和管理哲学到一个 AI Skill 中。

## 使用方法

| 命令 | 说明 |
|------|------|
| `/leader` | 调用领导 Skill 创建器 |

## 功能说明

Leader.skill 是一个 Claude Code 扩展，用于将领导者的愿景、决策风格和管理哲学蒸馏成一个 AI Skill。

### 工作流程

1. **录入信息**：称呼、基本信息、领导风格
2. **导入原材料**：会议录音、邮件、文档等
3. **分析生成**：提取愿景思维、决策风格、管理哲学
4. **确认写入**：生成 Leader Skill 文件

### 原材料类型

- 会议录音/纪要
- 邮件往来
- 文档/PPT
- 聊天记录
- 直接粘贴文字
