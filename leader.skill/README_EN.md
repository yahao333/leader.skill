# Leader.skill

Distill a leader's vision, decision-making style, and management philosophy into an AI Skill.

## Usage

| Command | Description |
|---------|-------------|
| `/leader` | Invoke the Leader Skill creator |

## Description

Leader.skill is a Claude Code extension that distills a leader's vision, decision-making style, and management philosophy into an AI-powered skill.

### Workflow

1. **Input Info**: Name, basic info, leadership style
2. **Import Materials**: Meeting recordings, emails, documents, etc.
3. **Analyze & Generate**: Extract vision thinking, decision style, management philosophy
4. **Confirm & Write**: Generate Leader Skill file

### Material Types

- Meeting recordings/notes
- Email exchanges
- Documents/PPTs
- Chat records
- Paste text directly
